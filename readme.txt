python3.X以上爬虫的学习

py_crawler01:一般来说这种简单的入门爬虫适合获取一些简单的内容，比如贴吧，论坛，模拟登陆，模拟点赞刷票之类的。如果是大型的项目，有可能有ajax之类的异步加载技术，需要做一些处理的。这时需要一些成熟的爬虫框架。
py_crawler02：一个爬取多个页面的png图像代码，由于API的变更，可能会有乱码的行为发生

py_crawler03：使用之前请确认已安装所需要的包，这个是用BeautifulSoup爬取淘宝首页的目录中的汉字，比re好用，基本不要写正则，也可以嵌套正则
